# less-for-web
A Blazor app that mimics some of the features of the `less` Linux command inside a Chrome Browser using the File System Access API so that you can peek inside a local file without reading the entire file.

<b>Try it out at: <a href="https://kristofferstrube.github.io/less-for-web/" target="_blank">https://kristofferstrube.github.io/less-for-web/</a></b>
